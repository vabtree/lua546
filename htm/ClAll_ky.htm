Graph Descriptions



Function Interface -- Graph shows (on left) public member functions of class calling (on right) other member functions of the class


Function and Variable Interface -- Graph shows (on left) global members of class calling, setting or reading (on right) other members of the class


Class Inheritance -- Cross reference graph shows immediate inheritance hierachy of class


Internal Function Calls -- Graph shows calling hierarchy among members of class


Internal Function and Variable Usage -- Graph shows function calls and variable use hierarchy among members of class


Member Functions Called from Other Classes -- Graph shows function calls made into members of the class, abstracted to the classes containing the functions


External Calls of Member Functions -- Graph shows (on left) external functions calling (on right) function members of class


External Use of Member Variables -- Graph shows (on left) external functions setting or reading (on right) variable members of class


Member Function Calls into Other Classes -- Graph shows function calls made from members of the class, abstracted to the classes containing the functions


External Functions Called -- Graph shows (on left) members of class calling (on right) functions outside of the class


External Variables Used -- Graph shows (on left) function members of class setting or reading (on right) variables outside of the class
