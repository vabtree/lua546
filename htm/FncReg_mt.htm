Metrics Descriptions



Lines in Function -- Number of lines in the function


Lines of Source Code -- Number of lines of code in the function


Returns -- 


Parameters -- 


Variables in Function -- Number of parameters and locals variables in the function.  To be accurate (non-zero), data must be collected with the -locals option enabled


Fan In -- Number of functions in the source files loaded into the project that call or read the function, either directly or through function pointers


Transitive Fan In -- Number of functions in the source files loaded into the project that transitively call or read the function, including calls through function pointers


Fan Out -- Number of functions that are called or read by the function, either directly or through function pointers


Transitive Fan Out -- Number of functions that are transitively called or read by the function, including calls through function pointers


Global Vars Used -- Number of global variables set and/or read by the function


Static Vars Used -- Number of static variables set and/or read by the function


Cyclomatic Complexity -- Cyclomatic complexity of the function (McCabe v(G)).  This represents the number of decision points in the function.  It also represents the number of linearly independent test paths through the function


Decision Density -- The decision density (cyclomatic density) of a function.  Calculated as the ratio of the cyclomatic complexity to the lines of statements for the function


Essential Complexity -- Essential complexity of the function (McCabe ev(G)).  This represents the number of decision points in the function that contain unstructured constructs


Essential Density -- The essential density, or degree of unstructuredness, of a function.  Calculated as the ratio of the essential complexity to the cyclomatic complexity


Program Difficulty -- Measure of how compactly the function implements its algorithms (Halstead D).  This is the inverse of the Halstead Program Level (of abstraction)


Program Volume -- Measure of the information content of the function (Halstead V)


Intelligent Content -- Language-independent measure of the amount of complexity of the function (Halstead I)


Mental Effort -- Measure of the number of elemental mental discriminations necessary to create, or understand, the function (Halstead E)


Decision Depth -- Number of nested levels of decisions or control statements within the function
